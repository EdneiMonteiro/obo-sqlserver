# Checklist de Publicacao

Antes de tornar o repositorio publico:

- [ ] Confirmar que o repositorio nao contem secrets, tokens, connection strings reais ou dumps.
- [ ] Revisar IDs de tenant/subscription e decidir se devem permanecer.
- [ ] Revisar logs, screenshots e exemplos.
- [ ] Confirmar que README, DISCLAIMER e SUPPORT deixam claro que e PoC.
- [ ] Confirmar que o modelo near-E2EE esta explicado.
- [ ] Confirmar LICENSE.
- [ ] Rodar build/test em ambiente com .NET 8 SDK.
- [ ] Rodar validacao Bicep.
- [ ] Confirmar que o cleanup remove todos os recursos.
